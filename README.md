# 📖 Introduction

This dashboard was created to help **Job Seekers, Career Changers, Recruiters, and Data Professionals** better understand the global data job market through interactive visualizations.

Using a real-world dataset of data-related job postings, this project transforms raw data into meaningful business insights using Power BI.

The dashboard enables users to:

- Analyze salary trends
- Compare job roles
- Explore hiring locations
- Identify hiring platforms
- Understand employment types
- Explore work-from-home opportunities
- Drill into individual job titles

# 📊 Key Insights

- Senior Data Scientist positions offer the highest median salaries.
- LinkedIn is the leading hiring platform.
- Full-time positions dominate the job market.
- North America and Europe contain the largest concentration of opportunities.
- Remote opportunities continue to represent a significant portion of openings.
- Specialized job roles generally command higher salaries.
